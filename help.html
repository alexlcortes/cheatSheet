Web Dev Cheat Sheet

Command Line Commands:

  • cd = "Change Directory"
  • ls = "List"
  • pwd = "Parent Working Directory"
  • mkdir = "Make Directory"
  • touch = "Make File"
  • cd .. = "Go Back One Folder"
  • tab = "Fills in file names"
  • up and down keys = "Will cycle through previous commands"

Git and GitHub.com Commands:

  • git status = "Gives you an update to your current work"
  • git clone = "Clones a repo created in github.com"
  • git add . = "adds new files to be commited"
  • git commit -m "" = "commits added files and allows for a message or comment"
  • git push origin master = "pushes commited changes to master branch"
  • git branch = "will show branches available"
  • git fetch = "fetch changes to branch"
  • git pull = "pulls master branch to your local branch"
  • git merge = "merges two branches together causing conficts to resolve"
